ExperienceGame Backend

Servidor API REST con **Node.js**, **Express** y **MongoDB** para gestionar usuarios, autenticación y publicaciones.

## Requisitos
- Node.js (v18+)
- MongoDB

## Instalación
```bash
npm install
```

## ⚙️ Variables de entorno (.env)
```bash
PORT=4000
MONGO_URI=mongodb://localhost:27017/experiencegame
JWT_SECRET=miclavesupersecreta
```

## ▶️ Ejecución
```bash
npm start
```
Servidor disponible en `http://localhost:4000/`
