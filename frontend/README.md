#  ExperienceGame Frontend

Interfaz web desarrollada en **Angular** para la aplicación *ExperienceGame*, que permite a los usuarios registrarse, iniciar sesión y crear publicaciones sobre videojuegos.

##  Requisitos previos
- Node.js (v18+)
- Angular CLI
- Visual Studio Code (opcional)

##  Instalación
```bash
npm install
```

## ▶ Ejecución
```bash
ng serve
```
Visita [http://localhost:4200](http://localhost:4200)

##  API
Configurada por defecto en `http://localhost:4000/api`
